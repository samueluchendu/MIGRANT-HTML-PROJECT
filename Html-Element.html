PROJECT ON 20 HTML ELEMENT

1. <header></header> Tag is used to group elements at the beginning of the page. example the navigation menu, the Banner of a webpage etc.

2. <i></i> Tag is used to italize a text. example the word Author is italized

3. <del></del> Tag is used to strike through a text to denote been deleted. example the word Author is deleted

4. <data></data> This gives a machine readable translation to a content when hovered upon, see example below:

  • Mini Ketchup
  • Jumbo Ketchup
  • Mega Jumbo Ketchup

5. <caption></caption> Tag is used to specify the caption or title of a table. see example below

This is a Caption

6. <bdo></bdo> Tag is used to override the current direction of a text it is used along with n attribute of dir, which can be either right to left (rtl) or left to right (ltr) example the word Author is spelt from right to left

7. <b></b> Tag is used to make a text bold. example the word Author is bold

8. <a></a> Tag is used to make a link, it uses an atrribute of href to specify the url. example the word this is a link

9. <abbr></abbr> Tag is used to make an accronym, it uses an optional atrribute of title to describe the accronym when hovered on. example the word HTML

10. <dfn></dfn> Tag is used to indicate the term been defined, example the word HTML is as a langauge that defines the structure of a webpage

11. <li></li> Tag is used to make a list, it is used either inside a ul tag or ol tag,See example below

  • First List
  • Second List

12. <ol></ol> Tag is used to a numbered a list of items, the li tag is used to display the content of the item,See example below

  1. First List
  2. Second List

13. <small></small> Tag is used to make the font size of a text one size smaller, example this is a small text

14. <sup></sup> Tag specifies inline text which is to be displayed as superscript, example 42 = 16

16. <sub></sub> Tag specifies inline text which is to be displayed as a subscript, example CH2

17. <var></var> Tag represents the name of a variable in a mathematical expression or a programming context, example let name is an example of Javascript variable declaration

18. <u></u> Tag underlines a word or text. example, the word Author is underlined

19. <samp></samp> Tag represents sample (or quoted) output from a computer program. example: Keyboard not found
Press F1 to continue
